## 如何使用？

找到SoftwareInc的安装文件夹，这通常可以再Steam中的SoftwareInc图标中点击`右键>属性>本地文件>浏览本地文件`就可以找到了。

打开文件夹后，游戏目录的结构类似于以下这样

```
Furniture/
Localization/
    English/
    ## 将本repo的SimplifiedChinese文件夹放在这里 ##
Materials/
```

进入游戏后在最左上角切换语言，完成！
